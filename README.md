# Future Ready Finance

**Live site:** https://scorrigan834-arch.github.io/future-ready-finance/

A free financial literacy website for people ages 14–35. Helps students, young adults, and early professionals make better financial decisions before the consequences become real.

## What's Inside

- **Life Stage Paths** — High School, College, Early Career, Establishing (ages 14–35)
- **Financial Decision Guides** — Student debt, apartment affordability, investing early, college decisions
- **Big Decisions** — Major financial choices organized by category
- **Simulations** — Model financial decisions before making them
- **Resources** — Frameworks, tools, and assessments

## Tech

Plain HTML and CSS. No frameworks, no build tools, no dependencies. Hosted free on GitHub Pages. Fonts from Google Fonts (Playfair Display + DM Sans).

## Audience

Students and young adults ages 14–35 making major financial decisions for the first time.
